Bật WLAN AutoConfig & Xem mật khẩu Wi‑Fi

Hướng dẫn chi tiết cho Windows 10/11 — giao diện siêu đẹp

⚠️ Chỉ làm với mạng bạn sở hữu

Tóm tắt

Nếu khi chạy netsh wlan show profiles gặp lỗi "The Wireless AutoConfig Service (wlansvc) is not running", bạn cần bật dịch vụ WLAN trước. Trang này hướng dẫn đầy đủ cách bật WLAN, xem profile và mật khẩu Wi‑Fi.

Bước 1 — Bật WLAN AutoConfig

  1. Nhấn Win + R → gõ services.msc → Enter.
  2. Tìm dịch vụ WLAN AutoConfig → chuột phải → Properties.
  3. Trong Startup type chọn Automatic → nhấn Start → Apply → OK.
  4. Nếu muốn nhanh bằng CMD: mở Command Prompt (Admin) và chạy:
    sc config wlansvc start= auto
sc start wlansvc

💡 Sau khi bật, chạy netsh wlan show profiles sẽ hiển thị các profile Wi‑Fi đã lưu.

Bước 2 — Xem mật khẩu Wi‑Fi

Sau khi thấy danh sách profile, nhập lệnh (thay Tên_WiFi bằng SSID thực):

netsh wlan show profile name="Tên_WiFi" key=clear

Thử nghiệm OpenAI API (tùy chọn)

Bạn có thể thử gọi OpenAI API trực tiếp từ trang này — không khuyến nghị do rủi ro lộ API key. Mình cung cấp 2 cách: Client-side (nhanh nhưng không an toàn) và Server-side (khuyến nghị).

Client-side (nhanh, KHÔNG AN TOÀN)

Cảnh báo: API key sẽ được gửi trực tiếp tới api.openai.com — key có thể bị lộ (CORS cũng có thể chặn request).


          


          

            
Server-side (KHÁM NGHIỆM) — Khuyến nghị

            
Luôn đặt API key trên server. Dưới đây là ví dụ Node.js/Express để bạn triển khai trên máy hoặc server riêng:

            
// server.js (ví dụ)
// npm i express node-fetch
const express = require('express');
const fetch = require('node-fetch');
const app = express();
app.use(express.json());

app.post('/api/openai-test', async (req, res) => {
  const prompt = req.body.prompt || 'Xin chào';
  try {
    const r = await fetch('https://api.openai.com/v1/chat/completions', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'Authorization': `Bearer ${process.env.OPENAI_API_KEY}`
      },
      body: JSON.stringify({ model: 'gpt-3.5-turbo', messages: [{role:'user', content: prompt}] })
    });
    const data = await r.json();
    res.json(data);
  } catch (e) { res.status(500).json({error: e.message}); }
});

app.listen(3000, ()=> console.log('Listening 3000'));

            
Sau khi triển khai, client chỉ gọi endpoint của bạn /api/openai-test, API key nằm an toàn trên server.

          

        

      


      

        
Bước 3 — Dùng giao diện Control Panel

        
Đi tới Control Panel → Network and Sharing Center → click tên Wi‑Fi → Wireless Properties → tab Security → tick Show characters.

        
Một số máy cần quyền admin để hiển thị mật khẩu.

      


      

        
⚠️ Cảnh báo an toàn

        
Không sử dụng hướng dẫn này để lấy mật khẩu Wi‑Fi của mạng mà bạn không được phép. Điều đó có thể vi phạm pháp luật.
